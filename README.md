# 📊 SubsTracker

**SubsTracker** es una aplicación diseñada para ayudarte a gestionar todas tus suscripciones de forma sencilla y organizada. Con ella podrás llevar un control de tus gastos mensuales y anuales, saber cuánto gastas en total y no olvidar cuándo se renueva cada servicio.

> 🚧 **Estado del proyecto:** En desarrollo activo.

---

## ✨ Características actuales

- ➕ Añadir nuevas suscripciones.
- ✏️ Editar suscripciones existentes.
- 🗑️ Eliminar suscripciones.
- 💰 Visualizar el gasto total de todas las suscripciones.
- 📈 Identificar la suscripción más cara.
- 📅 Ver la fecha de la próxima renovación de cada suscripción.
- 🌙 Cambiar entre modo claro y modo oscuro.

---

## 📷 Vista general

La aplicación cuenta con una interfaz sencilla e intuitiva donde encontrarás:

- Un resumen superior con:
  - 💸 Gasto total de todas las suscripciones.
  - 👑 La suscripción con el coste más elevado.
  - 🌗 Botón para cambiar entre tema claro y oscuro.
- Una lista de suscripciones donde cada una muestra:
  - Nombre del servicio.
  - Precio.
  - Tipo de pago (mensual o anual).
  - Próxima fecha de renovación.
  - Opciones para editar o eliminar.

---

## 🚀 Objetivo

SubsTracker nace con el objetivo de facilitar el control de las suscripciones digitales, permitiendo conocer en todo momento cuánto dinero se destina a ellas y cuándo será el próximo cobro.
